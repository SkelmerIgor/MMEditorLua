4.1 Имя в поиске должно быть указано строго на латинице
(( такое как “_” в поиске расценивается как МГ и не допускается к редактированию. Пример МГ: Fernando_Donizetti, Пример без МГ:
Victor Zubov. Так же такие ники как Vsem Privet, Yabloko Ananas, Vadim Stakan расценваются как МГ. ))
4.2 Заведомо-допустимые объявления:
- О поиске персонала/собеседовании для частных предприятий (мафии), если оно не находится под влиянием санкций СМИ.
- О поиске пиар-менеджера/сотрудников для бизнеса.
- О наборе игроков в БК, ФК, ЛК и т.д, если оно не находится под влиянием санкций СМИ (при указании GPS).
- О поиске второй половинки.
- О поиске партнёра для игры в Black Jack/кости
- О поиске автомехаников для АЗС.
- О заключении договора с АЗС. (от автомехаников).
- О поиске актеров.
- О поиске напарников для соревнований по стрельбе.
- О поиске тамады на свадьбу.
- О поиске людей по должности. ( исключение: лицензер, адвокат )
4.3 Не допустимые объявления:
- О поиске лицензера, адвоката.
- О себе: при встрече. (и.тд)
- О поиске самого себя, а также лиц, чье имя крайне похоже не ваше
Пример: человек под именем Vlad Kuzent ищет человека Vlad Kusent.
О поиске друзей, семьи
Пример: Ищу своих старых друзей/Ищу семью Remesnikov.
Примеры заведомо-правильных объявлений:
LS | Ищу человека по имени Artem Remesnikov. Просьба перезвонить.
LS | Проходит собеседование в БК “Grove Street”. GPS: 4-1.
LS | В магазин 24/7 требуются сотрудники.
LS | Познакомлюсь с девушкой для серьезных отношений.
LS | Заключу договор с АЗС.
LS | Ищу полковника Военно-Воздушных Сил.
LS | Ищу партнера для игры в кости. Звоните.
Примеры объявлений, которые не допускаются к редактированию:
LS | Идет набор в банду баллас. Ждем на районе.
( банда – БК, баллас – на латинице должен быть. Вместо ждем на районе должен быть указан GPS )
LS | Ищу девушку для серьезных отношений. О сесе: при встрече.
( Не ищу, а познакомлюсь. О себе: при встрече – не указываем )
LS | Ищу человека Alexandr.
( не указано полное имя )
