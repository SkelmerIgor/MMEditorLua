7.1 Ценовая политика для SIM-Card:
Минимальная цена для купли/продажи – 100$.
Максимальная цена для купли/продажи – 1.000.000$.
7.2 При продаже обязательное указание формата, пример:
Форматы: 2255 - AA-BB, 357735 - AB-CC-AB, 123123 - AB-CA-BC.
Примеры для редактирования:
Продам симку 2332 – LS | Продам SIM-Card формата “AB-BA”. Цена: договорная.
Примеры заведомо-правильных объявлений:
LS | Продам SIM-карту формата “AA-BC”. Цена: 890.000 $.
LS | Куплю SIM-карту. Бюджет: 300.000 $.
LS | Куплю SIM-карту формата AA-AA. Бюджет свободный.
LS | Обменяю SIM-card на формат «XXXX».
Примеры объявлений, которые не допускаются к редактированию:
LS | Продам SIM-карту. Цена: 890.000 $.
( не указан формат )
LS | Обменяю SIM-card на вашу.
( не указан формат )
